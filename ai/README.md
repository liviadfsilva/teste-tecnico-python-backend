# 🤖 Uso de IA no projeto

A IA foi utilizada como ferramenta de apoio para acelerar o desenvolvimento, revisão e melhoria de partes do projeto.

---

## 🧩 Áreas onde a IA foi utilizada

### 1. Cálculos de métrica

A IA foi utilizada como apoio para estruturar a lógica de cálculo das métricas de produtividade, garantindo clareza e legibilidade do código.

### 2. Estrutura do README
Auxílio na organização da documentação, deixando mais clara e adequada para apresentação em desafio técnico.

### 3. Design da API
Sugestões para separação de responsabilidades (controllers, models, schemas) e boas práticas de backend.

---

## ⚠️️ Importante

Todo o código e decisões foram revisados e adaptados manualmente para o contexto do projeto.

A IA foi usada como ferramenta de suporte, não como substituição de entendimento técnico.
